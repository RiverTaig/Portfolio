Running Calculator

Every serious runner will, from time to time, need to do a little (or a lot!) of math to find answers to some really basic questions. For example:

  • If I run a 3:15 marathon, what should my split be for 10km?
  • If I want to run at a pace of 7:30/mile, what speed in miles per hour (mph) is that on a treadmill?
  • If I can run 42:37 for a 10K, what would my predicted half marathon time be?
  • If I lost 10 pounds, how much faster could I run for 5K?
  • When training for a marathon, what is my 'lactate threshold' pace?
  • How does temperature, wind, elevation, climbing, and running on a treadmill with an incline affect my performance?

Now, you can find a million calculators on-line that will answer these questions, but I think you won't find the kind of one-stop-shopping you'll experience here.
